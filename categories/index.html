Get Maximum in Generated Array Given an integer n. An Array nums of length n+1 is generated in the following way: nums[0] = 0 nums[1] = 1 nums[2i] = nums[i], when `2 <= 2i <= n nums[2*i + 1] = nums[i] + nums[i+1], when 2 <= 2*i + 1 <= n Return the maximum integer in the array nums`. Solution def solve(n): if n < 2: return n x = [0] * (n + 1) x[1] = 1 for i in range(2, n): j, odd = divmod(i, 2) if odd: x[i] = x[j] + x[j+1] else: x[i] = x[j] result = max(result, x[i]) return result Kth Largest Element in an Array Given an integer array nums and an integer k, return the kth largest element in the array.
