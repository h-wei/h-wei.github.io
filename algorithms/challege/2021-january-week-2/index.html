Check If Two String Arrays are Equivalent

Given two strings s and t, return true if the two arrays respresent the same string, and false otherwise. A string is represented by an array if the array elements concatenated in order forms the string.

Solution

def solve(s: List[str], t: List[str]) -> bool:

Word Ladder

A transfomation sequence from word beginWord to word endWord using a dictionary wordList is a sequence of words beginWord -> s1 -> s2 -> ... -> sk, such that:

  • Every adjecent pair of words differs by a single letter.
  • Every si for 1 <= i <= k is in wordList. Note that beginWord does not need to be in wordList.
  • sk == endWord Given two words, beginWord and endWord, and a dictionary wordList, return the number of words in the shortest transformation sequence from beginWord to endWord, or 0 if no such sequence exists.

Intution

Solution

def solve(words: List[str], begin: str, end: str) -> int:
    if end not words:
        return 0

    aux = defaultdict(set)
    for w in words:
        for i in range(len(w)):
            key = w[0:i] + '?' + w[i+1:]
            aux[key].add(w)

    # bfs
    cand = deque([(begin, 0)])
    seen = {begin}
    while cand:
        word, depth = cand.popleft()
        for i in range(len(word)):
            key = word[0:i] + '?' + word[i+1:]
            for w in aux[key]:
                if w == end:
                    return depth + 1
                if w not in seen:
                    seen.add(w)
                    cand.append((w, depth+1))
    return 0

Create Sorted Array through Instructions

Given an integer array instuctions, you are asked to create a sorted array from the elements in instructions. You strat with an empty container nums. For each element from left to right in instructions, insert it into nums. The costof each insertion is the minumum of the following:

  • The number of elements currently in nums that are strictly less than instuctions[i].
  • The number of elments currently innums that are strictly greater than instructions[i]. For example, if inserting element 3 into nums = [1,2,3,5], the cost of insertion is min(2,1) (element1 1 and 2 are less than 3, element 5 is greater than 3) and nums will become [1,2,3,3,5]. Return the total cost to insert all elements from instructions into nums. Since tha answer may be large, return it modulo 10**9 + 7.

Solution

def solve(instructions: List[int]) -> int:
    MOD = 10**9 + 7
    result = 0
    aux = []
    for i, x in enumerate(instructions):
        left, right = bisect_left(aux, x), bisect_right(aux, x)
        result += min(left, i - right)
        aux[right:right] = [x]
        # aux.insert(right, x)
        # insort(aux, x)
    return result % MOD


class Bit:
    def __init__(self, n):
        self.__n = n
        self.__bit = [0] * (n + 1)

    def query(self, x):
        result = 0
        while x > 0:
            result += self.__bit[x]
            x -= -x & x
        return result

    def update(self, x):
        while x <= self.__n:
            self.__bit[x] += 1
            x += -x & x


def solve2(instructions: List[int]) -> int:
    MOD = 10**9 + 7
    n = max(instructions)
    bit = Bit(n)

    result = 0
    for i, n in enumerate(instructions):
        result += min(bit.query(n-1), i - bit.query(n))
        bit.update(n)
    return result % MOD

Merge Sorted Array

Given two sorted integer nums1 and nums2, merge nums2 into nums1 as one sorted array.

Intution

Solution

def solve(nums1: List[int], m: int, nums2: List[int], n: int) -> None:
    end = m + n
    while m > 0 and n > 0:
        if nums1[m-1] >  nums1[n-1]:
            nums1[end-1] = nums1[m-1]
            end, m = end-1, m-1
        else:
            nums1[end-1] = nums2[n-1]
            end, n = end-1, n-1
    while n > 0:
        nums1[end-1] = nums2[n-1]
        end, n = end-1, n-1

Add Two Numbers

Given two non-empty linked lists representing two non-negative integers. The digits are stroed in reversed order, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.

Intution

Solution

def solve(s: ListNode, t: ListNode) -> ListNode:
    c = dummy = ListNode()
    carry = 0
    while s or t or carry:
        if s:
            carry += s.val
            s = s.next
        if t:
            carry += t.val
            t = t.next
        carry, val = divmod(carry, 10)
        c.next = ListNode(val)
        c = c.next
    return dummy.next

Boats to Save People

Given an array people where people[i] is the weight of the ith person, and an infinite number of boats where each boat can carry a maximum weigth of limit. Each boat carries at most two people at the same time, provided the sum of the weight of those people is at most limit. Return the minimum number of boats to carry every given person.

Intution

Solution

def solve(people: List[int], limit: int) -> int:
    people.sort()
    result = 0
    start, end = 0, len(people) - 1
    while start <= end:
        if people[start] + people[end] <= limit:
            start += 1
        end -= 1
        result += 1
    return result

Minimum Operations to Reduce X to Zero

Given a positive integer array nums and an integer x. In one operation, you can either remove the leftmost or the rightmost element from the array nums and subtract its value from x. Note that this modifies the array fro future operations. Return the minimum number of operations to reduce x to exactly 0 if it’s possible, otherwise, return -1. Constraints

  • 1 <= nums.length <= 10**5

Intution

First, solving this problem using DP, but then realized that the constraints are too large. Then, thinking in reverse, that, instead of finding the minimum prefix + suffix, find the maximum subarray. Final, finding the maximum subarray is standard and can be done greedily.

Solution

def solve(nums: List[int], x: int) -> int:
    result = total = j = 0
    target = sum(nums) - x
    n = len(nums)
    if target < 0:
        return -1
    if target == 0:
        return n

    for i in range(len(nums)):
        total += nums[i]
        while total > target:
            total -= nums[j]
            j += 1
        if total == target:
            result = max(result, i - j + 1)

    return n - result if result else -1
