No 563. Chain of Blocks [Medium]

You are given a list blocks where each block contains two integers [start, end] where start < end. You can join two blocks if the end of one is equal to the start of another. Return the length of the longest chain of blocks.

Intution

  • Returning the length of the longest chain of blocks, so we used to consider it is a dyncamic programming question.
  • Joining two blocks if the end of one is equal to the start of another. so we sort it.

Soluction 1

def solve(blocks):
    # dp[i] = the length of the longest chain of block at i
    n = len(blocks)
    blocks.sort(key=lambda x: x[1])
    ends = [end for _, end in blocks]
    dp = [0] * n
    for i in range(n):
        left = bisect_left(ends, blocks[i][0], hi=i)
        right = bisect_right(ends, blocks[i][0], hi=i)
        dp[i] = 1 + (max([dp[j] for j in range(left, right)]) if left < right else 0)
    return max(dp) if dp else 0

Soluction 2

def solve(blocks):
    # dp[i] = the length of the longest chain of ending with i.
    blocks.sort(key = lambda x: x[1])
    dp = defaultdict(int) 
    for s, e in blocks:
        dp[e] = max(dp[e], dp[s] + 1)
    return max(dp.values()) if dp else 0

O(n) with one pass for-loop, and O(1) with dict put and get, in total, the time complixty is O(n).

All ends of the blocks is at most 2*n, so this is O(n) space.
